## 安裝

```bash
curl -sSf https://ai.finlab.finance/install.sh | sh
```

自動偵測你的 CLI（Claude Code / Codex / Gemini），安裝 [uv](https://docs.astral.sh/uv/)（如果沒有），然後設定 skill。
